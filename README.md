#🌤️ **Realtime Weather App**
A beautiful, responsive, and interactive weather application built with Vanilla JavaScript, HTML, and CSS. This app fetches real-time weather data from the OpenWeatherMap API and dynamically updates the UI such as background colour, humidity, wind status based on the current weather conditions.

#🛠️ **Technologies Used**
-> HTML5: Provides the semantic structure and layout of the application.
-> CSS3: Handles custom styling, flexbox layout, gradients, and UI polish.
-> JavaScript (ES6+): Powers the logic using DOM manipulation, async/await fetch requests, and event handling.
-> OpenWeatherMap API: The RESTful API used to fetch real-time global weather and climate data.
